Testing Tips

  • Use browser DevTools Network tab to see the redirect chain
  • Use curl -I -L to follow redirects and see all headers
  • Test from ProbeOps Terminal in different regions
  • Check if your browser caches 301/308 redirects (clear cache to re-test)

Example curl Commands

# Follow redirects and show headers
curl -I -L https://{{ ctx.host }}/r/301

# Show redirect chain without following
curl -I https://{{ ctx.host }}/r/301
