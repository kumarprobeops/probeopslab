About this lab

ProbeOps Lab is a public "reference edge" for testing redirects, caching directives, geo routing, and request headers.

It runs a minimal, fixed configuration so results are predictable and scriptable.

  • No cookies
  • No auth
  • No tracking

Maintained by ProbeOps (we use it internally for debugging too).

If you need to run the same checks from multiple geographic regions, automate them, or schedule monitoring, that's what ProbeOps is for: probeops.com
